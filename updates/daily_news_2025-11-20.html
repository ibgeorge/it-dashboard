Active Tickets Dashboard

The dashboard now displays tickets that agents are actively working on, helping managers:

  • See what's being worked on right now (updated in last 72 hours)
  • Identify long-running projects that need attention (over 40 days old)
  • Adjust monitoring thresholds on the fly using the settings panel
  • Auto-refreshes every 2 minutes to stay current

Smart Filtering System

Added powerful filters to focus on what matters:

  • Date Ranges: View current year, last 6/3/1 months, or all time
  • Anomaly Exclusion: Filter out tickets with unusual resolution times (projects, on-hold items)
  • Persistent Settings: Your filter choices are saved and remain active across pages
  • Different Defaults: Dashboard shows current year; Agent Performance shows all time
